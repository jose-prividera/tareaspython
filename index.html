⚙️ Configurar Google Sheets

Pasos para conectar:
1. Abrí tu Google Sheet → Extensiones → Apps Script
2. Pegá el código del archivo Code.gs incluido
3. Implementá como Aplicación web (Implementar → Nueva implementación)
   • Ejecutar como: Yo
   • Quién tiene acceso: Cualquier usuario
4. Copiá la URL generada y pegála abajo
5. Hacé clic en Guardar y probar conexión
